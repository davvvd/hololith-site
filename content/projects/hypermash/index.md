HyperMaSH introduces Vector-Time-Color Hyper Lithography, engineering intensity, polarization, time evolution, and wavelength together in a single holographic exposure. Using reversible azobenzene materials and holographic tools, it aims to directly produce high-resolution diffractive elements by using only light.
